# ROADMAP

* LiteRT Windows runtime
* GPU support
* SSE streaming
* OpenAI-compatible API
* Smart routing base
* Minimal Web UI
* VRAM-aware loading base
* Hot swap base

* VSCode/Cline integration refinement
